步驟一：下載 Tesseract 安裝程式
Tesseract 專案本身不直接提供 Windows 安裝檔，但他們推薦使用由德國曼海姆大學 (UB Mannheim) 維護的版本。

前往 Tesseract 在 UB Mannheim 的下載頁面： https://digi.bib.uni-mannheim.de/tesseract/

在這個頁面中，你會看到許多檔案。請找到最新版本的 64位元 (w64) 安裝程式。檔案名稱看起來會像這樣： tesseract-ocr-w64-setup-v5.x.x...exe (其中 v5.x.x 是版本號)。

點擊該檔案並下載。

步驟二：執行安裝程式並選擇語言
執行你剛剛下載的 .exe 檔案。

同意授權條款，然後點選 "Next"。

【關鍵步驟】選擇組件 (Select Components)：

在選擇組件的頁面，你會看到一個列表。請點開 "Additional language data"（額外語言資料）旁邊的 + 號。

向下捲動，務必勾選你需要的語言。

chi_tra (繁體中文 - Traditional Chinese)

chi_sim (簡體中文 - Simplified Chinese)

eng (英文 - English) (強烈建議勾選，因為許多模型會以英文為基礎)

你可以根據需求勾選任意多種語言。

選擇安裝路徑：

程式預設會安裝在 C:\Program Files\Tesseract-OCR。

請記住這個路徑，我們在下一步會用到它。建議保持預設路徑即可。

點選 "Next" 直到安裝完成。

步驟三：設定環境變數 (最重要的一步)
安裝完成後，你必須告訴 Windows 在哪裡可以找到 Tesseract 執行檔。這一步稱為「設定環境變數」。

開啟「編輯系統環境變數」：

點擊 Windows 的「開始」按鈕 (或按 Win 鍵)。

直接輸入「環境變數」或「Edit the system environment variables」。

在搜尋結果中，點選「編輯系統環境變數」。

在彈出的「系統內容」視窗中，點選右下角的「環境變數(N)...」按鈕。

編輯 Path 變數：

在下方的「系統變數(S)」區塊中，找到並選取 Path 這個變數。

點選「編輯(E)...」。

新增 Tesseract 路徑：

在「編輯環境變數」視窗中，點選右上角的「新增(N)」。

在反白突顯的空白行中，輸入或貼上你的 Tesseract 安裝路徑。

(如果你使用預設路徑) 請輸入： C:\Program Files\Tesseract-OCR

輸入完畢後，按 Enter 鍵。

(建議) 新增 TESSDATA_PREFIX 變數：

這一步是告訴 Tesseract 在哪裡尋找語言資料檔 (.traineddata)。

回到「環境變數」視窗（Path 的上一層）。

在「系統變數(S)」區塊中，點選「新增(N)...」。

在「變數名稱」欄位輸入： TESSDATA_PREFIX

在「變數值」欄位輸入你的 Tesseract 安裝路徑並在後面加上 \tessdata。

(如果你使用預設路徑) 請輸入： C:\Program Files\Tesseract-OCR\tessdata

點選「確定」。

最後，在所有開啟的設定視窗中，一路點選「確定」關閉所有視窗。

步驟四：驗證安裝
你必須開啟一個新的命令提示字元 (CMD) 或 PowerShell 視窗，舊的視窗不會載入你剛剛設定的環境變數。

按 Win + R，輸入 cmd，然後按 Enter。

在黑色的視窗中，輸入以下指令並按 Enter：

Bash

tesseract --version
如果你看到類似 tesseract v5.x.x... 的版本訊息，代表執行檔路徑已設定成功。

接著，輸入以下指令來檢查語言包：

Bash

tesseract --list-langs
你應該會看到一個列表。檢查列表中是否包含你在安裝時勾選的語言，例如 eng 和 chi_tra。

疑難排解： 如果系統顯示「'tesseract' 不是內部或外部命令...」，這 99% 是因為你的環境變數 Path 設定錯誤或沒有開啟新的 CMD 視窗。請回到步驟三仔細檢查路徑是否正確。

恭喜！你已經成功安裝 Tesseract-OCR。
