# Math Learning App README

本專案是一個使用 Flask 開發的數學學習應用程式

## Git 操作說明

當您在專案中完成了一些變更 (例如：修改程式碼、新增功能)，您可以使用 Git 將這些進度儲存起來，並推送到遠端的程式碼倉庫 (例如 GitHub, GitLab)。

### 將變更新增到暫存區

這個指令會告訴 Git 你想要將哪些檔案的變更納入下一次的提交中。

*   **新增所有變更的檔案**
    ```bash
    git add .
    ```


### 提交變更

這個指令會將您暫存的變更正式儲存到本地的 Git 歷史紀錄中。請務必撰寫一個有意義的提交訊息，簡要說明這次提交的內容。

```bash
git commit -m "在這裡寫下您的提交訊息，例如：修正登入頁面的排版問題"
```

### 推送到遠端主線

最後，這個指令會將您本地的提交推送到名為 `origin` 的遠端倉庫的 `main` (或 `master`) 分支上。這會讓其他協作者看到您的最新進度。

*   **推送到 `main` 分支**
    ```bash
    git push origin main
    ```
